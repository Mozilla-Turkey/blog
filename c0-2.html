C Öğreticisi 0.2 (Ortam Kurulumu)


Bu yazılar https://www.tutorialspoint.com/cprogramming/ adresindeki yazı dizilerinin çevirileridir.

Ortamınızı C programlama dili için ayarlamak istiyorsanız, bilgisayarınızda aşağıdaki iki yazılım aracına ihtiyacınız vardır: (a) Metin Editörü ve (b) C Derleyici.

Metin Editörü

Bu, programınızı yazmak için kullanılacaktır. Bazı editör örnekleri arasında Windows Not Defteri, İşletim Sistemi Düzenleme komutu, Brief, Epsilon, EMACS ve vim veya vi bulunur.

Metin editörlerinin adı ve sürümü farklı işletim sistemlerinde değişiklik gösterebilir. Örneğin, Not Defteri yalnızca Windows’ta kullanılabilir, Vim veya vi Linux veya UNIX’te de kullanılabilir. Editörünüzle oluşturduğunuz dosyalara kaynak dosyaları denir ve bunlar program kaynak kodlarını içerir. C programlarının kaynak dosyaları genellikle “.c” uzantısıyla adlandırılır.

Programlamaya başlamadan önce, bir metin editörünün bulunduğundan ve bir bilgisayar programı yazma, bir dosyaya kaydetme, derleme ve son olarak çalıştırma konusunda yeterli deneyime sahip olduğunuzdan emin olun.

C Derleyicisi

Kaynak dosyada yazılı kaynak kodu, programınız için okunabilir insan kaynağıdır. İşlemci tarafından verilen talimatlara göre programı gerçekten çalıştırabilmesi için makine dilinde “derlenmesi” gerekir.

Derleyici kaynak kodları çalıştırılabilir programlara derler. En sık kullanılan ve ücretsiz kullanılabilir derleyici GNU C / C ++ derleyicisidir, aksi takdirde ilgili işletim sisteminiz varsa, HP veya Solaris’ten derleyicileriniz olabilir.

Aşağıdaki bölümde GNU C / C ++ derleyicisinin çeşitli işletim sistemlerine nasıl kurulacağı açıklanmaktadır. C / C ++ ‘dan bahsetmeye devam ediyoruz çünkü GNU gcc derleyicisi hem C hem de C ++ programlama dilleri için çalışıyor.

UNIX/Linux’ta Yüklemek

Linux veya UNIX kullanıyorsanız, komut satırından aşağıdaki komutu girerek GCC’nin sisteminize yüklü olup olmadığını kontrol edin -

$ gcc -v

Makinenizde GNU derleyiciniz varsa, aşağıdaki gibi bir mesaj yazmalıdır -

Using built-in specs.

Target: i386-redhat-linux

Configured with: ../configure — prefix=/usr …….

Thread model: posix

gcc version 4.1.2 20080704 (Red Hat 4.1.2–46)

GCC kurulu değilse, https://gcc.gnu.org/install/ adresinde bulunan ayrıntılı talimatları kullanarak kendiniz yüklemeniz gerekecektir.

Bu ders Linux üzerinden yazılmıştır ve verilen tüm örnekler Linux sisteminin Cent OS’ta derlenmiştir.

Mac OS’ta Yüklemek

Mac OS X kullanıyorsanız, GCC’yi edinmenin en kolay yolu, Xcode geliştirme ortamını Apple’ın web sitesinden indirmek ve basit kurulum talimatlarını izlemektir. Xcode kurulumuna sahip olduğunuzda, C / C ++ için GNU derleyicisini kullanabileceksiniz.

Xcode şu anda http://developer.apple.com/technologies/tools/ adresinde mevcuttur.

Windows’ta Yüklemek

Windows’a GCC’yi kurmak için MinGW’yi yüklemeniz gerekir. MinGW’yi yüklemek için, www.mingw.org adresindeki MinGW ana sayfasına gidin ve MinGW indirme sayfasının bağlantısını izleyin.

MinGW- < version > .exe olarak adlandırılması gereken MinGW kurulum programının en son sürümünü indirin.

Min GW’yi kurarken, en azından, gcc-core, gcc-g ++, binutils ve MinGW çalışma zamanlarını yüklemelisiniz, ancak daha fazla yüklemek isteyebilirsiniz.

MinGW kurulumunuzun bin alt dizinini PATH ortam değişkeninize ekleyin, böylece bu araçları komut satırında basit adlarıyla belirtebilirsiniz.

Yükleme tamamlandıktan sonra, Windows komut satırından gcc, g ++, ar, ranlib, dlltool ve diğer birçok GNU araçlarını çalıştırabilirsiniz.
